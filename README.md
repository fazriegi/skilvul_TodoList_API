## List of Endpoints

### Auth

- `POST /auth/register`
- `POST /auth/login`

### Todos

- `POST /todos`
- `GET /todos`
- `GET /todos/:id`
- `PUT /todos/:id`
- `DELETE /todos/:id`
- `DELETE /todos`
